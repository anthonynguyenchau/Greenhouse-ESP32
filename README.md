This report documents the design, implementation, and operation of the Smart Greenhouse IoT System, an integrated Internet of Things (IoT) solution for real-time monitoring and automated environmental control. The system utilizes an ESP32 microcontroller to process data from distributed sensors measuring temperature, humidity, soil moisture, and light levels. Based on these inputs, control algorithms automatically trigger actuators such as pumps, fans, and humidifiers.
A centralized dashboard built with Node-RED provides visualization and manual control, while a Python server hosts an AI assistant powered by the Google Gemini API to answer plant care questions. Experimental results demonstrate that the system successfully maintains environmental parameters within user-defined thresholds with high reliability. The inclusion of MQTT communication ensures low-latency data exchange, making the system a viable solution for scalable precision agriculture.
